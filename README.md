# Paywall 收费插件 v4.2.1

> 作者：洛蒂 | 框架：AstrBot  
> 功能：群/私聊双维度计费 + 卡密充值 + 签到 + 商城 + 银行 + 红包

---

## 快速开始

1. 安装方式:在 AstrBot 插件市场中使用链接安装
2. 在 AstrBot 插件配置中填写参数
3. 重启或热重载

---

## 配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `cost_per_message` | 1.0 | 每条AI消息扣多少积分 |
| `free_user_quota` | 3000 | 新用户初始免费积分 |
| `free_group_quota` | 10000 | 新群初始免费积分 |
| `sign_min` / `sign_max` | 10 / 100 | 签到积分范围 |
| `sign_item_chance` | 20 | 签到掉物品概率(%) |
| `items` | `[]` | 道具商城预设 `"名称:价格"` |
| `general_items` | `[]` | 百货商城预设 `"名称:价格"` |
| `tax_rate` | 5 | 交易税率(%) |
| `bank_rate_min` / `bank_rate_max` | 1.0 / 2.5 | 银行日利率范围(%) |
| `admins` | `[]` | 管理员QQ号列表(字符串) |
| `user_whitelist` / `group_whitelist` | `[]` | 免费用AI的用户/群 |
| `enabled` | true | 总开关 |
| `contact_group` | `""` | 充值联系群号 |

## 核心机制

**双维度计费**
- 私聊：扣个人积分
- 群聊：扣群积分（全群共用）

**扣费流程**
1. 检查余额 → 不足则AI拦截并提示欠费
2. AI回复后 → 确认有内容才扣费（双重保险）
3. 插件指令不扣费

**管理员特权**：免费用、充值他人、生成卡密、全场打折、查看全站余额

## 常见问题

**Q：为什么没扣费？**  
A：可能是管理员/白名单、插件关闭、AI未回复、或余额已耗尽被拦截。

**Q：用户怎么充值？**  
A：①管理员手动 `/pw充值` ②生成卡密让用户自助 `/pw兑换` ③每天签到白嫖。

**Q：群积分用完怎么办？**  
A：管理员 `/pw群充值 群号 金额` 或群成员 `/pw群兑换 卡密`。

**Q：数据存储在哪？**  
A：AstrBot KV数据库，卸载插件不自动删除数据。
